Feature Set

Built For Colour And Motion

Vayu keeps routing, tone, space, and modulation in one place so the plugin feels fast in a real mix.

Five-Band EQ

  • Live response curve and analyzer
  • Fast shelf and bell shaping
  • Readable visual feedback

Eight Rack Modules

  • Distortion, chorus, flanger, phaser
  • Delay, stereo, reverb, compressor
  • Drag-to-reorder chain

Preset Workflow

  • Factory EQ and rack presets
  • User rack presets and effect patches
  • Quick recall without leaving the plugin
Install Paths

Where Vayu Goes

If you use the installer, this is where it should place the plugin files.

macOS

  • AU: /Library/Audio/Plug-Ins/Components
  • VST3: /Library/Audio/Plug-Ins/VST3

Windows

  • VST3: C:\Program Files\Common Files\VST3
  • Rescan plugins in your DAW after install

Linux

  • ~/.vst3
  • /usr/lib/vst3
  • /usr/local/lib/vst3
Interface
