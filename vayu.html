Modular Multi-Effect Plugin

Shape Motion Into Sound

Vayu is a plugin-first rack for sound design and mixing. It combines a five-band EQ with drag-to-reorder distortion, chorus, flanger, phaser, delay, stereo, reverb, and compressor modules in one polished DAW workflow.

Install Notes

Where Vayu Ends Up

The installers aim for the standard host scan locations. If you use the raw bundles instead, these are the paths you want.

macOS AU

  • /Library/Audio/Plug-Ins/Components
  • User-level alternative: ~/Library/Audio/Plug-Ins/Components

macOS VST3

  • /Library/Audio/Plug-Ins/VST3
  • User-level alternative: ~/Library/Audio/Plug-Ins/VST3

Windows VST3

  • C:\Program Files\Common Files\VST3
  • Rescan plugins in your DAW after install
